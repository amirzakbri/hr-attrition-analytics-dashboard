# 📄 Project Documentation

This directory contains the supporting documentation for the **HR Attrition Analytics Dashboard** project.

The documents provide business, technical, and analytical context for the project, allowing recruiters, hiring managers, and other analysts to understand the project beyond the dashboard itself.

---

## Documentation Overview

| File | Purpose |
|------|---------|
| **01_Business_Requirements.pdf** | Defines the business objectives, project scope, stakeholders, business questions, and success criteria. |
| **02_Data_Dictionary.xlsx** | Describes every dataset field, data type, business definition, and value mapping used throughout the project. |
| **03_Project_Methodology.pdf** | Documents the complete analytical workflow from raw data collection through cleaning, transformation, dashboard development, and reporting. |
| **04_Portfolio_Case_Study.pdf** | Presents the project as a professional portfolio case study, highlighting the business problem, analytical process, insights, recommendations, and lessons learned. |

---

## Purpose

The documentation was created to demonstrate a complete end-to-end analytics workflow rather than only presenting the final dashboard.

It reflects the documentation standards commonly used in business intelligence and data analytics projects.
